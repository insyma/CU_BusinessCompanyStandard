Responsive Web Design (RWD)
The cost benefits are clear: one version of the website to build and to maintain.

Thanks to the *Responsive Web Design (RWD) there is no need for the separate mobile versions. This relatively new approach to web design allows building websites and newsletters that are optimized for screens of all sizes: small mobile devices, tablets and large desktop monitors.

Same counts for Newsletters, thanks to the Responsive Design they will automatically adapt to the screen size and capabilities of any device.
